JAAG: a JSON input file Assembler for AlphaFold 3 with Glycan integration

JAAG is a web-based tool designed to facilitate the assembly of input JSON files for standalone AlphaFold 3.

How to add glycans?
  1. Protein: 'Detect sequons' or add 'Glycosylation sites'
  2. Ligand: 'Glycan' is the default input type
Where to find GlycoCT?
  1. Draw a glycan in a popup SugarDrawer
  2. External glycan database: GlyGen, GlyTouCan, GlyConnect, etc
Reference
  1. Please refer to this paper to properly interpret the AlphaFold 3 glycan-related models:
    2. Huang, C., Kannan, N., & Moremen, K. W. (2025). Modeling glycans with AlphaFold 3: capabilities, caveats, and limitations. Glycobiology, 35(10), cwaf048
  2. If you use JAAG in your research, please cite:
    3. Huang, C., & Moremen, K. W. (2025). JAAG: a JSON input file Assembler for AlphaFold 3 with Glycan integration. bioRxiv
  3. JAAG source code and detailed guide can be found on JAAG GitHub and AlphaFold 3 GitHub
